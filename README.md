## 📖 إيه هو النظام ده؟

نظام **نقاط بيع متكامل** للمحلات التجارية. تقدر تبيع، ت manage المخزون، ت track الحسابات، وتطلع تقارير — **كل حاجة في مكان واحد**.

**مينفعش كلام فاضي:** ده نظام شغال فعلاً ومنتج جاهز للتشغيل.

## 🎯 ليه تختار النظام ده بدل غيره؟

| المشكلة في الأنظمة التانية | الحل في نظامنا |
|---------------------------|----------------|
| تدفع 1000-3000 ريال شهرياً | **اشتري مرة واحدة** واستخدم مدى الحياة |
| مشتغلكش من غير إنترنت | **شغال من غير إنترنت** - لو النت قطع، بتكمل بيع عادي |
| بياناتك عند الشركة مش عندك | **البيانات عندك** - على جهازك الشخصي |
| صعب التعديل والتغيير | **الكود عندك** - تقدر تعدل اللي تحب |
| مش بيدعم العربي كويس | **عربي 100%** - واجهة، تقارير، حسابات |
| كل سنة يطلبوا فلوس عشان يحدثوه | **التحديثات مضمونة** - طول عمر الرخصة شغالة |
| ياخد نسبه من كل عملية بيع | **ولا قرش** - كل أرباحك لك |

## ✨ إيه اللي تقدر تعمله؟

### 🛒 البيع
- تبيع بسرعة: اكتب اسم المنتج أو امسح الباركود
- طرق دفع: كاش، كارت، آجل
- مرتجعات واستبدالات
- طباعة فاتورة حرارية

### 📦 المخزون
- تعرف كمية كل منتج في المخزن
- تنبيه لو المنتج خلص أو قرب يخلص
- أوامر شراء واستلام
- طباعة باركود للمنتجات
- أسعار مختلفة (قطاعي، جملة، VIP)
- خصومات وعروض ترويجية

### 📒 الحسابات (الجزء اللي المحاسب بيحبه)
- **دليل حسابات كامل** - 16 حساب جاهزين
- **قيد محاسبي مزدوج** - مدين ودائن
- **رصيد افتتاحي** - أدخل الرصيد لما تضيف حساب جديد
- **مصروفات** - كل المصروفات مسجلة
- **إدارة الخزينة** - متابعة الصندوق

### 📊 التقارير
- تقرير يومي: مبيعات، مرتجعات، مصروفات
- مقارنة مبيعات: أسبوع/شهر/سنة
- أفضل وأسوأ المنتجات مبيعاً
- تقارير العملاء: مين أكثر واحد اشترى
- أرباح وخسائر مع رسومات بيانية
- **النشاط**: كل حصة اتعملت في النظام مسجلة وممكن ترجع فيها

### 🔒 الأمان والتدقيق
- **سجل تدقيق** - كل عملية مسجلة: مين، إيه، إمتى
- **تراجع (Undo)** - لو غلطت، ارجع بضغطة زر
- **إعادة (Redo)** - لو غيرت رأيك

---

## 🚀 ازاي تشتغل؟

1. **حمل** البرنامج من أعلى الصفحة
2. **ركبه** - التثبيت مباشر وسهل
3. **افتحه** - الداتابيز بتتعمل أوتوماتيك والبيانات الافتراضية بتظهر
4. **ابدأ بيع** - دوّر على المنتج وابدأ

**التحديثات:** لما ننزل إصدار جديد، البرنامج يتحدث نفسه بنفسه. بس طفيه وشغله تاني وهتكون أخر نسخة.
